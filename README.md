## 项目概述


## 技术栈

- 后端：Python 3.7、Flask
- 数据库：MySQL 5.7
- 前端：Vue

## 功能模块

### 管理员功能

- 用户管理
- 员工管理
- 商品分类管理
- 超市商品管理
- 商品进货管理
- 供应商管理
- 系统管理
- 订单管理
- 用户资料管理

### 员工功能

- 超市商品操作
- 商品进货管理
- 订单管理
- 用户资料操作

### 前台用户功能

- 商品浏览
- 购物车管理
- 我的订单
- 个人中心服务

## 系统角色

- 管理员：负责系统全面的操作与维护
- 员工：负责日常商品、订单和用户管理的操作

## 运行环境

- 操作系统：兼容性良好，可在多数操作系统上运行
- 后端环境：Python 3.7 或以上版本
- 数据库环境：MySQL 5.7 或以上版本
- 前端环境：现代浏览器

## 部署步骤

1. 环境准备：确保 Python 3.7、MySQL 5.7、Node.js（Vue）已安装
2. 数据库搭建：根据提供的 SQL 脚本创建数据库及表结构
3. 后端部署：配置 Flask 应用并运行
4. 前端部署：编译 Vue 项目并部署到 Web 服务器
